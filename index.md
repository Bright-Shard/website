# What is this website for?
I'm BrightShard, a teenager who is developing FOSS software - that is, Free and Open-Source Software. I programmed this website to share my apps with the world! If you're wondering, yes, this website is also FOSS. You can see it's code [here.](https://github.com/Bright-Shard/bright-shard.github.io/)

### What does FOSS mean?
FOSS, as I said above, stands for Free and Open-Source Software. It's easier to explain it word-by-word:
- **Free:** My apps have no cost, no ads, and no trackers! As the consumer, you will get none of the ads or trackers other apps press onto you.
- **Open-Source:** "Ok, sure, you can say that about your apps; but do you have evidence?" you might say. And the answer is yes, because my apps are open-source on my [GitHub Page.](https://github.com/Bright-Shard/) Open-source just means you can see and modify the code yourself! You can scan every line of code and you will not find a single ad, nor a single tracker. Pretty awesome, eh?
- **Software:** Software is software, there's not much to explain here. All of my apps are software, and so is my website.

# Where do I get your apps?
Unfortunately, I don't have a release for my apps right now. I'm still developing them and have several features done, but not enough that I feel like either of them are ready for release. You can still go to their web pages, though, and find out more about what I will be releasing!

- **Hypertasker:** Hypertasker will help you organize your life, with note-taking, email, calendar/to-do, and RSS support - all in one app. Instead of opening multiple browser tabs, a few apps, and spending several minutes trying to organize everything, you can just open Hypertasker and manage it all from one place.
- **Shell Striker:** [Ethical Hacking](/what-is-ethical-hacking.md/) is a sort of side hobby of mine. So I'm making Shell Striker - a program inspired by Armitage to automate all of the nitty gritty tasks in Pentests and Capture the Flags.

These are the two apps I'm working on right now. I will work on, release, and update these for months to come! Again, these will both be FOSS, and you can help contribute to them!

# How do I contribute?
My software is open-source on GitHub, which means you can not only see the code but also suggest edits! Just clone the project, make your changes/additions, and make a pull request. You can find my GitHub profile [here,](https://github.com/Bright-Shard/) and find more information on GitHub pull requests [here.](https://docs.github.com/en/github/collaborating-with-issues-and-pull-requests/about-pull-requests/)

If you're not into coding, but still want to help, you can just open an issue! Go to my profile, find the project you found a bug on or want to suggest a feature for, go to issues, and just open a new one. Be sure to be **as specific as possible** so that others can help most efficiently.
