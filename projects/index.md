## What is this page?
This page is for my projects. You can see them on the left navigation bar of this page.

## What projects do you make?
I'm currently making Hypertasker, a hub for many small but useful tasks, and Shell Striker, a hacking GUI inspired by Armitage. You can find out more about them by going to their project page.

## How do I get your projects?
Right now, I don't have a release for any of my projects. When they are available, there will be a "Releases" section in the navigation bar.

## Can this run on [computer name here]?
My apps will run on *all* computers (This includes Linux!). For Windows, just get the app installer. On MacOS, there will be a .app file to run. On Linux, there will be an AppImage file to run. You will need GTK for it to work.

## How do your apps run everywhere?/I want to make an app!
I use an **AWESOME** program called Beeware. Beeware lets you make apps in Python and then run them on anything - including mobile, although mobile has limited support, hence why my apps don't work on it (Yet - once there are more Beeware updates, my apps will be compatible!) Even better, Beeware is completely open-source on GitHub, meaning you can report bugs or suggest your own code! You can check out Beeware, a tutorial on how to use it, and it's GitHub on it's page in the sidebar to the left.
