## What is Beeware?
[Beeware](https://beeware.org) is a suite of FOSS Python tools to make and deploy Python apps on every operating system - Linux, MacOS, Windows, and even mobile(Android, iPadOS, and iOS)!

## How does Beeware work?
Beeware has their own UI library, [Toga](https://toga.readthedocs.io/), which is used to make an app. This is then used with their [Briefcase](https://briefcase.readthedocs.io/) library to cross-compile the app into another programming language that runs natively on the machine's operating system and produce an executable app which can be run there. This works for MacOS, Windows, Linux (With GTK), and Android as well. Beeware also works with iOS and iPadOS, however, because Apple is a jerk about their ecosystem, you have to have MacOS to build apps for iOS/iPadOS.

## How do you make an app with Beeware?
First, you should know at least a bit of Python. You can use some Google-fu to find a tutorial - just make usre it's Python3, not Python2, because Python2 is outdated. Then go to their [tutorial](https://docs.beeware.org/en/latest/tutorial/tutorial-0.html) to learn how to make an app with it. Afterwards, you can go to their [Toga documentation](https://toga.readthedocs.io/) page to learn the rest of the Beeware suite.
